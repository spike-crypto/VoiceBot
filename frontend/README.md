# Voice Bot Frontend

React frontend for the Voice Bot Interview Assistant.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
```

3. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build

```bash
npm run build
```

## Features

- WebRTC audio recording
- Real-time conversation display
- Audio playback
- Session management
- Error handling
- Responsive design
